# Typescript-Football-Data

Practice project working on nested JSON football data in TypeScript.

## Concepts Covered

- Type interfaces and intersection types
- flatMap(), filter(), reduce(), sort()
- Command line arguments

## Data

- squad_data.json - Premier League dquad stats
- match_events.json - match timeline events
- champion_league.json - UCL group stage data
- transers.json - summer 2024 transfer window

## Running 
- npm install
- npx ts-node src/squad.ts
- npx ts-node src/transfers.ts Arsenal
